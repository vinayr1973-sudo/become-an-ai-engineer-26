# Phase 3 — Build the Harness Yourself (Weeks 10-13)

**The highest-leverage phase in the roadmap.** You will not make the right
production trade-offs until you've built a harness once. See `SKILL.md` →
*Phase 3* for the 10 components.

**The 10 components:** loop control · tool dispatch · context management
(auto-compaction at 85%) · persistence · sub-agent orchestration · skills /
progressive disclosure · hooks · observability (OpenTelemetry) · sandboxing ·
auth brokering.

**Deliverable:** a ~1,500-line mini-harness in Python, plus a 1,000-word
post-mortem comparing it to the Claude Agent SDK and Deep Agents — what you got
right, what you cut, what you'd do differently.

> `code/mini_harness/` is yours to write. Don't copy a framework — build it,
> then read the frameworks and see what they hid from you.
